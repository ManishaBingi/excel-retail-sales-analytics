# Excel Retail Sales Analytics

##  Overview
This project demonstrates advanced Excel analysis using a large retail dataset.  
It includes formulas, pivot tables, and dashboards for business insights.

##  Files
- `Retail_Sales_Data.xlsx` → Raw dataset (10,000+ rows)
- `Analysis.xlsx` → Analysis file with formulas, pivot tables, dashboards
- `README.md` → Documentation

##  Features
- Advanced formulas: VLOOKUP, INDEX-MATCH, COUNTIFS, SUMIFS
- Pivot tables: Region-wise sales, product category analysis, top customers
- Dashboard: KPIs, charts, slicers for interactive analysis

##  How to Use
1. Open `Retail_Sales_Data.xlsx` for raw data
2. Explore `Analysis.xlsx` for formulas and pivot tables
3. Use the dashboard to filter and visualize insights
