# Excel Superstore Sales Analytics

## Overview
This project uses the Kaggle Superstore dataset to demonstrate advanced Excel analytics.  
It showcases formulas, pivot tables, and dashboards to uncover business insights.

## Files
- `Superstore_Data.xlsx` → Raw dataset (Orders, Customers, Products)
- `Superstore_Analysis.xlsx` → Analysis file with formulas, pivot tables, dashboards
- `README.md` → Documentation and insights

##  Features
- Advanced formulas: VLOOKUP, INDEX-MATCH
- Pivot tables: Region-wise sales, product category analysis, top customers etc.,
- Dashboard: KPIs, charts, slicers for interactive analysis
- Business storytelling: Linking raw data to actionable insights

## How to Use
1. Open `Superstore_Data.xlsx` for raw data
2. Explore `Superstore_Analysis.xlsx` for formulas and pivot tables
3. Use the dashboard to filter and visualize insights
